The function Ackermann(m, n) computes a result for two positive integers m and n using these rules:

  • when m = 0, the answer is n + 1
  • when m > 0 and n = 0, the answer is the result of the Ackermann function using arguments m-1 [as m] and 1 [as n]
  • when both m > 0 and n > 0, the answer is the result of the Ackermann function using arguments m-1 [as m] and the results of Ackermann(m, n-1) [as n]

Write a recursive implementation of the Ackermann function that accepts two integers and returns a integer result, and write a main function to allow the user to input two initial values.
