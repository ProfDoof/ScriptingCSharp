Created a program to convert length measurements from metric to English units. Prompt the user to enter a length in feet and inches. Use a function float convert(float feet, float inches) to return the number of centimeters in the given number of feet and inches.

For this problem, assume there are 2.54 centimeters in an inch.

Sample runs are shown below.

Feet? 1
Inches? 0
The length is 30.48 cm.

Feet? 1
Inches? 1
