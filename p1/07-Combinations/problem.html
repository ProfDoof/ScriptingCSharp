Create a function, Combinations(int n, int k) to calculate the number of ways to select k items from n items, commonly spoken as "n choose k".

The value of "n choose k" can be determined by using Pascal's Triangle, where n is the row, beginning with 0, and k is the position in the row, beginning with 0 and ending at k. For example, "4 choose 2" is 6, since item number 2 (the third one) on row number 4 (the bottom row) is 6.

The values of Pascal's Triangle can ge determined recursively as shown in the animating image on the right. Each item is either a base case (1), or the sum of two numbers on the previous row.


Running your program should look like this:

Enter an Integer: 6
Enter another Integer: 2
Combinations(6,2) = 15

Enter an Integer: 8
Enter another Integer: 4
Combinations(8,4) = 70
