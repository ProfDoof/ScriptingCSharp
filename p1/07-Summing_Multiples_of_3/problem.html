Write a program that, given an integer number, calculates and prints the sum of all positive multiples of 3 between zero and number (including number if it is a multiple of 3).

You must write and use a recursive function sum3s that accepts one integer parameter and returns the sum of the positive multiples of 3 between zero and that value. Your function should work for any integer.

Your program should run like the examples shown below:

Enter number: 10

The sum is 18.

In the first example (above): the positive multiples of 3 between 0..10 are {3,6,9}, which sum to 18.

Enter number: 12

The sum is 30.

In the second examle (above): the positive multiples are {3,6,9,12}, which sum to 30.

Enter number: 4

The sum is 3.

Enter number: 1

The sum is 0.
