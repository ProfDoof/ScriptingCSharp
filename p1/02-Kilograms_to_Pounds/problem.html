The goal of this exercise is to practice using data types and various manipulations of data types.

Create a program that converts kilograms into English pounds and ounces. You should prompt the user for kilograms and display the equivalent pounds and ounces. Two example runs are shown below. 

This program converts kilograms to pounds and ounces.
Kilograms: 1

1 kilograms is 2 pounds and 3.2 ounces.

This program converts kilograms to pounds and ounces.
Kilograms: 8.25

8.25 kilograms is 18 pounds and 2.4 ounces.

For this problem, assume a kilogram is 2.2 pounds.
