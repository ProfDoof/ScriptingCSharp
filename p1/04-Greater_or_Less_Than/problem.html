Create a program that accepts two integers as input and prints a statement to the screen indicating if the first number is greater or less than the second. Your program should run like the samples shown below.

This program determines the relationship between two input numbers.
Enter the first integer: 5
Enter the second integer: 2
5 is greater than 2

This program determines the relationship between two input numbers.
Enter the first integer: 7
Enter the second integer: 8

For this assignment, you may assume that the two numbers will never be equal.
