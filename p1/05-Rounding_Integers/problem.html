Create a function int round(int number, int unit) that will round number down to the nearest multiple of unit. For example, rounding 5127 down to a unit of 10 becomes 5120.

Create a program to test your function that allows the user to enter a number, then calls the function and displays results using rounding units 10, 100, and 1000.

Two sample runs are shown below.

Enter an integer: 5127

Round to ten: 5120
Round to hundred: 5100
Round to thousand: 5000

Enter an integer: 893

Round to ten: 890
Round to hundred: 800
Round to thousand: 0
