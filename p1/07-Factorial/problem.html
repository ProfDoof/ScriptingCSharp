In mathematics, the factorial of a positive integer is the product of all integers 1..n, so n! = n * (n-1) * (n-2) * ... * 2 * 1. Also, as a special case, 0! = 1.

Write a function int fact(int n) that calculates and returns the factorial of n. In your function, you may assume that n is non-negative (zero or greater than zero).

Create a main function that prompts the user to enter a number, then uses the fact function to calculate the factorial value, which should then print to the screen.

Your program should run like the examples shown below (user input shown in red):

This program calculates factorials.
Enter an integer: 1

1! = 1

This program calculates factorials.
Enter an integer: 3

3! = 6

This program calculates factorials.
Enter an integer: 4

4! = 24
