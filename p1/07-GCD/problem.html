Write a program that uses the Euclidean algorithm to calculate the greatest common divisor (GCD). For two integers a and b, gcd(a,b) is:

  • a, if b = 0
  • gcd(b, a mod b), if b != 0

Create a recursive function int gcd(int, int) that accecpts two integers and returns the greatest common divisor between them. Also, create a main function to accept input and show the results of the function.

Running your program should look like the example below:

This program calculates the greatest common divisor (GCD) for two integers.

Enter a number: 6
Enter another: 8

GCD = 2


This program calculates the greatest common divisor (GCD) for two integers.

Enter a number: 24
Enter another: 30

GCD = 6
