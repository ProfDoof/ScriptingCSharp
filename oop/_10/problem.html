The purpose of this assignment is to practice C++ class implementation allowing value semantics for a simple class: IntegerSet, an unordered container of int values.

In addition to the previous container requirements, your class should provide all functionality required to allow an instance of IntegerSet to be used as a value-type. That is, instances (values) should be copyable, assignable, passable, initializable, and displayable as if they were a value of a built-in type just like an int.

In addition to canonical form, your code should allow values of type IntegerSet to be formatted and output using iostream; again, this is just like values for ints.

The following code is a partial example of how your class will be tested.
