The purpose of this assignment is to practice C++ class implementation for a simple class: IntegerQueue, an ordered container of int values.

Given an integer value N and an IntegerQueue q, the following operations should be defined:

q.push(N)appends N to back of ints in q
q.pop()removes and returns N from the front of ints in q
q.empty()returns true iff q contains no ints
q.size()returns the number of ints contained by q

Your implementation should meet the following additional criteria:

  • use a dynamic array of ints, since there is no maximum size specified
  • separate the implementation (.cpp) from the interface (.h)
  • compile without any warnings when using the following options:
    -Wall -pedantic-errors -Werror -Wfatal-errors -Wextra
    -Wshadow -Wno-unused-variable -Wno-unused-but-set-variable
  • using namespace must not occur in any header
  • allow non-mutating functions to operate on const objects

The following program is a trivial example of how your class will be tested.

#include <iostream>
#include "IntegerQueue.h"
using namespace std;

int main()
{
    IntegerQueue q;
    cout << (q.empty() ? "q is empty":"q is not empty") << endl;
    q.push(3);
    q.push(4);
    q.push(2);
    cout << q.pop() << endl;
    cout << "q.size = " << q.size() << endl;
    cout << (q.empty() ? "q is empty":"q is not empty") << endl;
}
