The purpose of this assignment is to practice C++ class implementation for a simple class: IntegerSet, an unordered container of int values.

Given an integer value N and an IntegerSet s, the following operations should be defined:

s.contains(N)returns true iff N is one of the ints contained in s
s.insert(N)adds N to the set of ints contained in s
s.remove(N)removes N from the set of ints contained in s
s.empty()returns true iff s is the empty set
s.size()returns the cardinality of the set s (or |s|)

Your implementation should meet the following additional criteria:

  • use a dynamic array of ints, since there is no maximum size specified
  • separate the implementation from the interface
  • compile without any warnings

The following program is a trivial example of how your class will be tested.

#include <iostream>
using namespace std;
#include "IntegerSet.h"

int main()
{
    IntegerSet s;
    cout << (s.empty() ? "s is empty":"s is not empty") << endl;
    s.insert(3);
    s.insert(4);
    s.remove(2);
    s.remove(3);
    
    cout << "|s| = " << s.size() << endl;
    cout << (s.contains(4) ? "s contains 4":"s does not contain 4") << endl;
    cout << (s.contains(3) ? "s contains 3":"s does not contain 3") << endl;

}
